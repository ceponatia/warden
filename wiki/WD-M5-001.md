# WD-M5-001 -- Deep import into package internals

## What this means

Code is importing private internal paths from another package/module.

## Common causes

- Missing public export surface
- Convenience import bypassing package boundary contracts

## Resolution patterns

### Pattern: Export from package root
Promote needed symbols to official entry points and update callers.
