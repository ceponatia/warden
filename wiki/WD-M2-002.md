# WD-M2-002 -- Stale file not imported

## What this means

A file is stale and appears unreferenced, suggesting dead code risk.

## Common causes

- Deprecated implementation left behind
- Feature rollback without cleanup

## Resolution patterns

### Pattern: Delete or allowlist intentionally retained files
Remove dead code; if retained for reference, add explicit allowlist entries.
