# WD-M1-001 -- File growth exceeds threshold

## What this means

A file is growing faster than the repository baseline for the selected window.

## Common causes

- Feature accretion in a single module
- Missing extraction/refactor boundaries

## Resolution patterns

### Pattern: Extract cohesive submodules
Split responsibilities into smaller modules with narrow interfaces.
